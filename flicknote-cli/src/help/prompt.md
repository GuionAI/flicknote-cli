Examples:
  flicknote prompt add --title "My Prompt" --prompt "You are a ..."
  flicknote prompt list
  flicknote prompt detail <prompt-id>
  flicknote prompt modify <prompt-id> --title "New Title"
  flicknote prompt modify <prompt-id> --prompt "Updated prompt"
  flicknote prompt delete <prompt-id>

Prompts can be associated with projects using `flicknote project modify`.
