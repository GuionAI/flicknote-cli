Examples:
  flicknote keyterm add --name "My Terms" --content "term1: definition"
  flicknote keyterm list
  flicknote keyterm detail <keyterm-id>
  flicknote keyterm modify <keyterm-id> --content "updated content"
  flicknote keyterm delete <keyterm-id>

Keyterm sets can be associated with projects using `flicknote project modify`.
